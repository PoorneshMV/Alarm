Alarm App

A modern Alarm Clock application built using SwiftUI and MVVM Architecture.

Features

* Create and manage multiple alarms
* Enable or disable alarms with a toggle
* Edit alarm time and settings
* Local notification support
* Clean and responsive SwiftUI interface
* Light and Dark Mode support
* Splash Screen and onboarding flow

Technologies Used

* SwiftUI
* MVVM Architecture
* UserNotifications Framework
* Swift
* Xcode

Requirements

* macOS
* Xcode 16 or later
* iOS 18+ Simulator or Physical Device

Installation

Option 1: Clone the Repository

git clone https://github.com/PoorneshMV/Alarm.git

Navigate to the project folder:

cd Alarm

Open the project in Xcode:

open Alarm.xcodeproj

If the project contains a workspace file:

open Alarm.xcworkspace

Option 2: Download ZIP

1. Click Code → Download ZIP on GitHub.
2. Extract the ZIP file.
3. Open the project folder.
4. Double-click the .xcodeproj or .xcworkspace file.

Running the App

1. Open the project in Xcode.
2. Select an iPhone Simulator or connected device.
3. Press ⌘ + R to build and run the application.
